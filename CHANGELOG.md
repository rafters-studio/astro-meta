# @rafters/astro-meta

## 0.0.0

Initial scaffold. Package shape, public types, no runtime implementations.

The xEO emission plane for Astro 6: JSON-LD, llms.txt, robots, sitemap, OG
images, and build-time GEO readability scoring. Per-surface subpath exports.
Composes with `@rafters/astro-data` on the data plane and `eavesdrop` on the
observation plane.
